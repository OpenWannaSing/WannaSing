# 音乐播放功能 - 产品需求文档

## 文档属性

| 属性 | 内容 |
|------|------|
| 功能 | 音乐播放器 |
| 版本号 | V1.0 |
| 文档状态 | 初稿 |
| 目标平台 | Web |

## 1. 功能概述

WannaSing 的音乐播放功能，提供专业级的音频播放体验，支持多种音乐格式、播放控制、进度管理、音量调节等功能。

## 2. 核心功能

### 2.1 播放控制

| 功能 | 说明 |
|------|------|
| 播放/暂停 | 点击按钮控制播放状态 |
| 进度控制 | 拖动进度条跳转到任意位置 |
| 上一首/下一首 | 在播放列表中切换（后期） |
| 停止播放 | 立即停止音频播放 |

### 2.2 音量控制

| 功能 | 说明 |
|------|------|
| 音量滑块 | 拖动调节音量大小 |
| 静音切换 | 一键静音/取消静音 |
| 音量图标 | 直观的音量状态显示 |

### 2.3 播放模式

| 模式 | 说明 |
|------|------|
| 单曲循环 | 重复播放当前歌曲 |
| 列表循环 | 循环播放整个列表（后期） |
| 随机播放 | 随机选择歌曲（后期） |

### 2.4 音频信息

| 信息 | 说明 |
|------|------|
| 歌曲名称 | 显示正在播放的歌曲 |
| 艺术家/来源 | 显示创建者或来源 |
| 时长显示 | 当前时间/总时长 |
| 播放模式指示 | 显示当前播放模式 |

### 2.5 视觉效果

| 效果 | 说明 |
|------|------|
| 播放波形动画 | 随音乐跳动的可视化 |
| 旋转唱片动画 | 播放时唱片旋转效果 |
| 进度条动画 | 平滑的进度过渡 |
| 按钮悬停效果 | 交互反馈动画 |

## 3. UI/UX 设计

### 3.1 播放器布局

```
┌─────────────────────────────────────────────────┐
│  🎵  [唱片旋转动画]  歌曲名称                    │
│              [艺术家信息]                       │
│  ◀◀   [进度条]    ▶▶                            │
│     [播放/暂停]  [单曲循环]  [音量滑块]         │
└─────────────────────────────────────────────────┘
```

### 3.2 交互设计

#### 播放按钮
- 状态：未播放 → 三角形图标
- 状态：播放中 → 两竖线图标
- 点击：切换播放状态，并有缩放动画

#### 进度条
- 拖动：平滑动画
- 悬停：显示时间提示
- 点击：立即跳转

#### 音量控制
- 滑块：从左到右音量递增
- 点击静音：一键切换
- 图标随音量变化

## 4. 技术实现

### 4.1 技术栈
- HTML5 Audio API
- Web Audio API (可视化)
- Framer Motion (动画)
- React Hooks (状态管理)

### 4.2 组件结构

```
MusicPlayer
├── CoverArt (唱片封面/旋转效果)
├── ProgressBar (进度条)
├── Controls (播放/暂停/上一首/下一首)
├── VolumeControl (音量控制)
└── Waveform (音频可视化)
```

### 4.3 状态管理

```typescript
interface PlayerState {
  isPlaying: boolean;
  currentTime: number;
  duration: number;
  volume: number;
  isMuted: boolean;
  loopMode: 'none' | 'one' | 'all';
  currentTrack: Track | null;
}
```

## 5. 动画效果

### 5.1 唱片旋转
- 播放时：持续旋转动画
- 暂停时：减速停止
- 时间：2秒/圈，平滑过渡

### 5.2 按钮反馈
- 悬停：放大 + 颜色加深
- 点击：缩放动画
- 图标切换：淡入淡出

### 5.3 进度条
- 平滑过渡：CSS transition
- 播放时：渐变色彩流动

## 6. 错误处理

| 场景 | 处理 |
|------|------|
| 音频加载失败 | 显示错误提示 + 重试按钮 |
| 网络中断 | 暂停播放 + 显示提示 |
| 格式不支持 | 提示用户格式问题 |
