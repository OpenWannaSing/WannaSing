## 3. 核心服务设计

### 3.1 音高分析服务

**文件：`/services/pitch_service.py`

**职责：**
- 加载音频文件
- 检测基频 (F0)
- 计算音域统计量
- MIDI 转换

**算法：**
- PYIN 算法（Librosa 实现）
- 静音过滤
- 统计分析

**输入输出：**
```python
{
    "lowest_note": int,          # 最低音 MIDI
    "lowest_note_name": str,      # 最低音音符名
    "highest_note": int,         # 最高音 MIDI
    "highest_note_name": str,     # 最高音音符名
    "vocal_range": int,           # 音域宽度
    "mean_pitch": float,         # 平均音高
    "pitch_variation": float,   # 音高变化
    "max_jump": float            # 最大音程跳
}
```

### 3.2 节奏分析服务

**文件：** `/services/rhythm_service.py`

**职责：**
- 检测 BPM
- 节拍跟踪
- 切分音计算
- 节奏类型分类

**技术方案：
- Librosa beat_track
- onset_strength 分析

### 3.3 结构分析服务

**文件：** `/services/structure_service.py`

**职责：**
- 段落边界检测
- 段落分类（前奏/主歌/副歌）
- 结构可视化

### 3.4 难度评分服务

**文件：** `/services/difficulty_service.py`

**职责：**
- 多维度评分
- 权重计算
- 等级判定

```
本地开发
├── 前端：Vite 开发服务器 (localhost:5173)
├── 后端：Uvicorn (localhost:8000)
└── 数据库：SQLite (/.wannasing/wanasing.db)
```
